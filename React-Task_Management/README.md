# React Task Manager
-Created a Task Manager to learn to use Hooks in React like:
    -useState
    -useContext
    -createContext
    -useCallback
    -useMemo
    -useRedeucer

 -Task Manager performs basic features like:
  - Add a new task
  - Delete a task
  - Mark a task complete or incomplete
  - Check total number of completed task
